# Heart Disease Risk Classifier

A Streamlit web application that predicts heart disease risk using machine learning.

## Features
- Patient data input form
- XGBoost model prediction
- Risk probability calculation
- SHAP explanations (optional)

## Installation

```bash
git clone <your-repo-url>
cd heart_disease-ml-project
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

## Model
- Algorithm: XGBoost
- Input: Patient demographics and medical metrics
- Output: Heart disease risk probability
